# App Bar Component

Usually consists of a title and a left button, sometimes a right button.

## Examples

``` js
<AppBar>
  <Left>
    <IconButton><MdMenu className="white f3" /></IconButton>
  </Left>
  <Title>AppBar Simple</Title>
  <Right>
    <FlatButton><MdMoreVert className="white f3" /></FlatButton>
  </Right>
</AppBar>
```
